# 📋 朝会・夕会：新ダッシュボードの使い方

**🎯 明日から使える！デジタル朝会・夕会システム完全ガイド**

---

## 🌅 朝会での使い方

### **📱 朝会前の準備（5分前）**

1. **ダッシュボードを開く**
   ```
   https://your-domain.com/dashboard
   ```
   - 📱 タブレットの場合：ブラウザを横向きで使用すると見やすいです
   - 💻 PCの場合：そのまま使用OK

2. **朝会タブをクリック**
   - 画面上部の「朝会」タブをクリック
   - 自動的に朝会モードに切り替わります

---

### **👥 司会者の進行方法**

#### **🎯 司会ガイドページを活用**
1. **専用司会ページを開く**
   ```
   https://your-domain.com/meeting-guide
   ```
   または、ダッシュボードの「🎯 司会ガイド」ボタンをクリック

2. **画面の見方**
   - **上部**: 現在時刻とモード（朝会/夕会）表示
   - **中央**: 今日のサマリー情報
   - **下部**: 進行メモとクイックアクション

#### **📝 進行手順**
1. **安全確認**
   - 画面の「安全HUD」セクションで昨日の報告事項を確認
   - 重要なアラートがある場合は最初に共有

2. **進捗確認**
   - 「記録進捗」で各モジュールの入力状況をチェック
   - 遅れているものがあれば担当者に確認

3. **優先対応者確認**
   - 「優先対応者」リストで今日の重要タスクを確認
   - 各担当者に簡単にヒアリング

4. **引継ぎ事項**
   - 「引継ぎサマリー」で前回からの継続事項を確認

---

### **⚡ スタッフ個人の活用方法**

#### **🎯 NextActionCardを活用**
朝会中に表示される「次にやるべきこと」カードを確認：

1. **カードの見方**
   - **優先度**: 緊急度に応じた色分け表示
   - **時間目安**: 所要時間の表示
   - **内容**: 具体的なアクション

2. **そのまま仕事に移行**
   - カードをクリック→関連画面に自動ジャンプ
   - 例：
     - 日常記録入力 → 個人支援画面
     - スケジュール調整 → スケジュール管理画面
     - 週間確認 → 週間スケジュール画面

3. **効率的な使い方**
   - 朝会終了後、すぐにカードをクリック
   - 画面が切り替わったら、そのまま作業開始
   - **紙のメモを取る必要がありません**

---

## 🌆 夕会での使い方

### **📱 夕会前の準備**

1. **ダッシュボードを開く**（朝と同じURL）
2. **夕会タブをクリック**
   - 自動的に夕会モードに切り替わります
   - 一日の振り返り情報が表示されます

### **👥 司会者の夕会進行**

#### **🎯 司会ガイドページ活用**
夕会モードでは以下の流れで進行：

1. **一日の実績確認**
   - 記録入力の完了状況をチェック
   - 未入力がある場合は翌日対応を確認

2. **安全報告**
   - 一日の中で発生した安全に関わる事項
   - 明日への申し送り事項

3. **明日の準備**
   - NextActionCardで明日の優先事項を確認
   - 特別な予定や注意事項の共有

## 💡 よくある質問・トラブルシューティング

### **Q: タブレットで文字が小さい**
**A:** ブラウザの拡大機能を使用
- Safari/Chrome: 画面をピンチアウト（指2本で広げる）
- 推奨倍率：110-125%

### **Q: NextActionCardが表示されない**
**A:** 以下を確認
1. 朝会タブが選択されているか
2. ページを更新してみる（F5またはリロード）
3. インターネット接続を確認

### **Q: クリックしても画面が切り替わらない**
**A:**
1. 少し待ってからもう一度クリック
2. 画面を更新してからクリック
3. それでも解決しない場合は管理者に連絡

### **Q: 司会ガイドページが開かない**
**A:**
- URLを直接入力するか、ダッシュボードの「🎯 司会ガイド」ボタンをクリック
- 新しいタブで開くので、元のダッシュボードも見続けられます
