# `/kiosk/users` 別事業所試行（初期データ作成）テンプレート

紙記録からのデータ化は次の雛形を用い、法人側の確認を取りながら作成する。

実利用者名・個人情報はリポジトリに含めない。

## 利用者マスタ雛形（users-template.csv）

```csv
利用者表示名,利用者ID,所属事業所,有効状態,支援手順記録対象,紙記録の保管場所,主記録の扱い,更新日,入力者,確認者,承認者,紙記録との対応
```

## 支援手順マスタ雛形（procedures-template.csv）

```csv
手順ID,利用者ID,手順名,実施タイミング,本人の動き,支援者の手順,注意事項,禁忌事項,更新日,作成者,確認者,承認者,紙記録参照
```

## 役割分担

- 入力者: 紙記録から転記
- 確認者: 現場実態との適合性を確認
- 承認者: 試行対象として可否を決定

## 試行対象化判定（3〜5名）

- 対象外理由がある場合は「対象外」と明記
- 人物特定情報は別紙運用し、本体にはプレースホルダ管理

## 監査記録（開始前）

| 利用者ID | 利用者名(別紙参照) | 手順ID | 試行可否 | 入力者 | 確認者 | 承認者 | 保留理由 |
| --- | --- | --- | --- | --- | --- | --- | --- |
|  |  |  |  |  |  |  |  |
|  |  |  |  |  |  |  |  |

## 注意

- 禁忌・注意事項は空欄不可
- 更新日・確認者・承認者が未入力は対象外
- 紙記録参照が不明確な項目は対象外
