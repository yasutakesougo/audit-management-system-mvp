# 📋 朝会・夕会 司会者用台本

## 🌅 朝会進行（08:50-09:10）

### 事前準備（1分）

1. **MeetingGuideページを開く** (`/meeting-guide`)
2. **朝会モードを選択**（画面上部で自動判定されているか確認）
3. **Drawerを開く**（右上のハンバーガーメニュー）

### 進行台本（15-20分）

#### ① 申し送り確認（5分）

**「昨日の申し送りを確認します」**

- Drawer内の「申し送りミニサマリー」を確認
- 件数を読み上げ：「昨日は○件の申し送りがありました」

**重要案件がある場合（赤バッジ）:**

```
「重要案件が○件あります。○○さんの件について共有します」
→ 詳細をクリックして内容を読み上げ
→ 「担当者の方、確認をお願いします」
```

**要注意案件がある場合（黄バッジ）:**

```
「要注意案件が○件あります。○○の件、気にかけてください」
```

#### ② 今日の予定確認（5分）

**「今日の予定を確認します」**

- スケジュール画面で今日の予定をざっと確認
- 特別な予定があれば共有

#### ③ 利用者別の注意事項（5分）

**「利用者別に気をつけることはありますか？」**

- 参加者から口頭で共有
- 重要な事項があればその場で申し送りタイムラインに記録

#### ④ その他・終了（2分）

```
「他に共有事項はありますか？」
→ あれば共有
「今日もよろしくお願いします」
```

## 🌆 夕会進行（16:30-17:00）

### 事前準備（1分）

1. **MeetingGuideページを開く**
2. **夕会モードを選択**
3. **Drawerを開いて今日のサマリーを確認**

### 進行台本（25-30分）

#### ① 今日の申し送り確認（10分）

**「今日の申し送りを確認します」**

- 「今日は○件の申し送りがありました」
- カテゴリ別件数を読み上げ：「体調○件、行動面○件...」

**未対応案件の確認:**

```
「未対応の案件が○件あります。確認していきます」
→ 1件ずつ読み上げて状況確認
→ 対応済みのものは「対応済み」に更新
```

**重要案件のフォロー:**

```
「重要レベルの案件についてフォローします」
→ 各担当者に状況を確認
→ 必要に応じて明日への引き継ぎ事項を確認
```

#### ② 今日の気づき追加（10分）

**「今日気づいたことで申し送り事項はありますか？」**

- 各スタッフから共有
- その場で申し送りタイムラインに入力
- 重要度の判定を確認

#### ③ 明日への引き継ぎ（5分）

```
「明日特に注意することはありますか？」
→ 重要な案件は「重要」レベルで記録
→ 担当者の引き継ぎ事項確認
```

#### ④ 終了（2分）

```
「お疲れさまでした」
「日次記録の作成をお願いします」
```

## 🎯 司会者のコツ

### 📱 システム操作

- **状態更新**: 対応済みの案件はその場で「対応済み」に変更
- **新規入力**: 会議中に出た新しい申し送りはその場で記録
- **重要度判定**: 迷った時は「要注意」レベルで記録

### 🗣️ 進行のコツ

- **時間管理**: 各セクションの時間を意識して進行
- **参加促進**: 発言の少ないスタッフにも声かけ
- **記録確認**: 重要な内容は復唱して確認

### 🚨 トラブル対応

**システムが開けない場合:**

```
「システム不具合のため、従来通り口頭で進行します」
→ 後でシステム管理者に連絡
→ 重要事項は紙にメモして後で入力
```

**操作がわからない場合:**

```
「操作については後で確認します。まず口頭で共有しましょう」
→ 会議後にサポート担当者に確認
```
