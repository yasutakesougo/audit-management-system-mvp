# 【クロノート】本番運用スタートガイド（現場・管理者用）

本システムを安全に使い始めるための「最低限これだけ」のルールです。

---

## 📅 1. 出欠・実績の入力（毎日）
- **操作:** 登所・退所時刻、ステータスを入力して「保存」します。
- **ポイント:** 設定が見当たらない、または「保存できない」場合は、管理者に **「SharePoint の列設定（複数選択可）」** の確認を依頼してください。
- **消去:** 一度入れた値を「空」にしても消えない場合は、備考欄に「入力ミス」等の注記をお願いします。

---

## 📝 2. 業務日誌・支援記録（毎日）
- **重要ルール:** 同じ日の日誌を複数人で書く場合は、**「保存する直前にブラウザの更新（F5）ボタン」** を押して、他の人の書き込みを取り込んでから保存してください。
- **最終保存:** その日の終わりには、**特定の担当者（リーダー等）が最終確認** をして保存することを推奨します。

---

## 📋 3. 個別支援計画（ISP）
- **動作:** アセスメントや計画設計は JSON (データ一括保持) で管理されているため、大量に文章を書いても動作が重くなりにくい設計です。
- **安心:** 保存ボタンを押した後、一度画面を閉じてから開き直し、内容が正しく残っているか初回のみ指差し確認をお願いします。

---

## ⚠️ 困ったときは？
1. **画面が真っ白になった / 動かない**
   - ブラウザを一度閉じ、再起動してもう一度試してください（Fail-Open 設計により、データが消えるリスクは最小限に抑えられています）。
2. **保存ボタンが押せない**
   - ネットワーク接続（SharePoint へのアクセス）を確認してください。
3. **エラーが消えない**
   - 管理者に連絡してください。その際、**「キーボードの F12 キー」** を押して出る赤い文字の画面をスマホ等の写真で共有いただけると解決が早まります。
