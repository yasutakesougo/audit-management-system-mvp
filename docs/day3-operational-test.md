## 補足: Dev Panel 活用ポイント

各フェーズで FAIL が出た場合の即時調査手順:

| 症状 | Dev Panel での調査 |
|------|------------------|
| 画面が白い | `Ctrl+Shift+D` → Lists → リスト存在確認 |
| データが表示されない | SELECT タブ → 該当リスト + `$select=*` → データ有無確認 |
| Create が失敗 | Fields タブ → Required 確認 → POST タブで最小 JSON テスト |
| 型エラー | OpeningVerification → Step2 → type_mismatch 確認 |
| 「Field not found」 | Fields タブ → InternalName 確認 → FIELD_MAP と照合 |
