# ダッシュボード使い方ガイド

## はじめに

このガイドでは、**3つの基本画面** を覚えるだけで、システムを効果的に活用できる方法をご案内します。

## 基本の3画面 📱

### 📍 普段使い：🏠 今日の全体状況

**いつ使う？** → 日中の業務中に情報を確認したいとき

**何が見える？**

- 今日のSafety HUD（安全状況の概要）
- 予定の重なり状況
- 記録の進捗状況
- 緊急アラート（あれば）

**どう使う？**

1. システムを開いたらまずここを見る
2. 赤いアラートがあったら優先対応
3. 予定の重なりがあったら職員同士で相談

---

### 📍 朝の準備：☀ 朝のミーティング

**いつ使う？** → 9:00頃の朝会で

**何が見える？**

- 今日の安全確認事項
- 優先的にフォローしたい利用者さん
- 朝会チェックリスト
- 今日の全体スケジュール

**どう使う？**

1. 朝会が始まったらこの画面に切り替え
2. 安全確認事項を全員で確認
3. チェックリストを順番に進める
4. フォロー対象の利用者さんを担当者で分担

---

### 📍 一日の振り返り：🌙 夕方のミーティング

**いつ使う？** → 17:15頃の夕会で

**何が見える？**

- 今日の記録完了状況
- 利用者さんの健康状態サマリ
- 夕会チェックリスト
- 明日への申し送り事項

**どう使う？**

1. 夕会が始まったらこの画面に切り替え
2. 記録が未完了のものを確認
3. 健康状態で気になる点を共有
4. チェックリストで漏れがないか最終確認
5. 明日のフォロー事項を確認

---

## 迷ったときの基本ルール 🧭

| 時間帯 | どの画面を開く？ | 主な目的 |
|--------|------------------|----------|
| **日中業務** | 🏠 今日の全体状況 | 情報確認・業務進行 |
| **朝会（9:00頃）** | ☀ 朝のミーティング | 一日のスタート準備 |
| **夕会（17:15頃）** | 🌙 夕方のミーティング | 一日の振り返り・明日準備 |

## よくある質問 ❓

**Q: 他のタブ（医療・健康、支援記録など）はいつ使うの？**
A: 基本は上記3つだけでOKです。他のタブは専門業務や詳細分析が必要なときに使用します。

**Q: 予定の重なりが表示されたらどうする？**
A: まずは関係職員同士で相談。システムが時間や担当者の代替案を提案してくれる場合もあります。

**Q: チェックリストの項目をクリックしても大丈夫？**
A: はい！クリックすると完了マークがつきます。取り消しも可能です。

**Q: タブレットやスマートフォンでも使える？**
A: はい。画面サイズに合わせて自動調整されます。

## 新人さんへのメッセージ 💪

### 覚える基本

「この3画面だけ覚えれば大丈夫」

まずは日中の 🏠 今日の全体状況 に慣れて、朝会・夕会のときだけタブを切り替える、というシンプルな使い方から始めてください。

慣れてきたら他の機能も少しずつ活用していきましょう。
