# 行政監査（実地指導）対応ガイド：モニタリング編

本システムは、厚生労働省の定める強度行動障害支援の基準に基づき設計されています。監査員への説明時には、以下のポイントに沿ってシステムの信頼性を提示してください。

## 1. 監査員への基本説明（スクリプト）

> 「当事業所では、強度行動障害支援のモニタリング業務をシステム化し、制度要件（3ヶ月ごとの実施、研修要件、支援計画との整合性）を自動的にチェックしています。会議記録は作成後にシステム上で『確定』され、改ざんができない状態で保管されています。本日提示する帳票は、すべてこの確定済みデータから出力されたものです。」

---

## 2. 重点確認ポイントと回答案

### Q. 会議が本当に行われたか、後からまとめて作っていないか？
- **回答**: 「システムには履歴（FinalizedAt）が日時秒単位で記録されています。ダッシュボード上でも実施状況がリアルタイムに表示されており、未実施や期限超過は即座に発見し是正する運用を行っています。」
- **提示エビデンス**: 制度遵守ダッシュボードの「モニタリング検出」履歴

### Q. 会議の参加スタッフの研修要件は満たされているか？
- **回答**: 「会議記録ごとに、参加したスタッフが『基礎研修』『実践研修』を修了しているかをシステムが自動判定しています。資格不備がある場合は、保存時に警告が出るとともに、監査ダッシュボードにアラートが表示されるため、不合格な会議がそのまま放置されることはありません。」
- **提示エビデンス**: 会議詳細画面の「資格チェック：適合」表示

### Q. 支援計画（シート）に基づいた検討が行われているか？
- **回答**: 「会議記録は、必ずその時点での有効な『支援計画シート』を選択して作成する仕様になっています。計画内容とモニタリング結果が紐付いているため、PDCAサイクルが分断されることはありません。」
- **提示エビデンス**: PDF帳票の上部に印字されている「対象支援計画名」

---

## 3. 監査当日の提示手順

1. **全体像の提示**:
   - 「制度遵守ダッシュボード」を見せ、事業所全体のモニタリング充足状況が「グリーン（適切）」であることを示す。
2. **個別証跡の提示**:
   - 特定の利用者の「モニタリング会議記録」を開き、検討内容の具体性を確認してもらう。
3. **帳票の提出**:
   - 必要に応じて、確定印（日時・確定者名入り）が印字されたPDFを印刷して提出する。

---

## 4. 運用の自己点検（チェックリスト）

- [ ] 毎月、ダッシュボードを見て「期限超過」の利用者がいないか確認しているか。
- [ ] すべての会議が「確定（Finalized）」済みになっているか。
- [ ] スタッフマスタの資格情報（研修修了履歴）が、最新の状態に更新されているか。
