# ブリッジ機能 概要

> アセスメントの情報を、支援計画シートと手順書兼記録へ安全に取り込み、
> その根拠・出典・履歴まで追跡できるブリッジ機能です。

---

## 説明責任の4層

| 層 | 問い | 実現手段 |
|:---:|---|---|
| **①** | 取り込めるか？ | assessmentBridge / planningToRecordBridge |
| **②** | なぜ入ったか？ | provenance（変換理由 + 出典ラベル） |
| **③** | どこに反映されたか？ | フィールド横バッジ / [📋 計画] バッジ |
| **④** | いつ誰が入れたか？ | ImportHistoryTimeline |

---

## 安全性

| 特性 | 説明 |
|---|---|
| **追記マージ** | 既存データを上書きしない |
| **冪等性** | 同じデータは重複しない |
| **手動編集保護** | 職員が書いた内容は消えない |
| **プレビュー** | 取込前に内容を確認できる |
