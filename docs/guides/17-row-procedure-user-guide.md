# 17行標準支援手順 運用マニュアル

## 1. 17行標準支援手順とは
「17行標準支援手順」は、利用者一人ひとりの一日の流れを17の項目（行）に整理した共通フォーマットです。
個別支援計画で設計した「本人の動き」や「支援の具体策」を、日々の記録（日次記録）へ自動的に反映させることで、現場での支援の質と記録の整合性を高めることを目的としています。

## 2. 計画から記録への反映の流れ
個別支援計画シート（L2）の「支援設計」タブで入力した手順は、日次記録（L3）の「支援手順パネル」へ自動的に引き継がれます。

### 構造化手順がある場合
支援計画シートで「ステップ追加」を行い、行番号（1〜17）を指定して入力した内容は、そのまま日次記録の該当行に反映されます。
*   **本人の動き（活動詳解）** → 支援手順パネルの「本人」
*   **支援者の支援（手順詳解）** → 支援手順パネルの「支援」
*   **留意事項（様子・条件）** → 支援手順パネルの「留意事項」

### 構造化手順がない場合（レガシー集約）
まだ手順が構造化されていない古い計画書の場合でも、以下の内容が特定の日中活動行に集約して表示されます。
*   **対応方針** → 「AM日中活動」の本人の動き
*   **具体策** → 「AM日中活動」の支援者の支援
*   **環境調整** → 「PM日中活動」の留意事項

## 3. 支援手順パネル（ProcedurePanel）の見方
記録画面に表示される手順パネルには、以下の情報が表示されます。

| 項目名 | 内容 |
| :--- | :--- |
| **時間・活動名** | 9:30頃 通所・朝の準備、10:20〜 AM日中活動 など |
| **本人** | 利用者がその時間帯に具体的にどのような動きをするか |
| **支援** | 職員がどのような声かけ、見守り、介助を行うか |
| **留意事項** | 環境設定や注意点、成功のための条件など |

### AM/PM 日中活動と外活動
*   **日中活動（5行目、10行目、12行目）**: 室内での活動や通常のプログラム内容が表示されます。
*   **外活動（16行目、17行目）**: 公園、散歩、買い物などの外活動がある場合に使用します。準備と活動本体に分かれています。

## 4. 職員が確認すべき観点
記録を入力する際、以下の点を確認してください。

1.  **17行が表示されているか**: 欠けている行がないか確認します。
2.  **順番が崩れていないか**: 朝から帰りまでの流れが正しい順序になっているか確認します。
3.  **現場実態と合っているか**: 表示されている手順が、現在の利用者の状態や実際の支援内容と乖離していないか確認します。
    *   ※乖離がある場合は、支援計画の見直し（モニタリング・修正）を検討してください。

## 5. FAQ
*   **Q: 支援計画で入力したのに反映されません。**
    *   A: 支援計画シートが「保存」されているか、また対象の利用者が正しいか確認してください。
*   **Q: 行番号（1〜17）は必ず指定しないといけませんか？**
    *   A: 推奨されます。行番号を指定しない場合は、活動名や時間帯から自動推測してマッピングされますが、行番号指定が最も確実です。
