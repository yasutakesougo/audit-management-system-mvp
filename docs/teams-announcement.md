## 📋 簡単な使い方

### **🌅 朝会参加者（全員）**

1. **ダッシュボードを開く**

   ```text
   https://your-domain.com/dashboard
   ```

2. **朝会タブをクリック**

3. **NextActionCard（次にやること）を確認**
   - 会議後、カードをクリック→そのまま作業開始！
   - メモ取り不要です 📝✨

### **👥 司会者の方**

1. **司会ガイドページを開く**

   ```text
   https://your-domain.com/meeting-guide
   ```

   または、ダッシュボードの「🎯 司会ガイド」ボタンをクリック

2. **画面に沿って進行**
   - 安全確認 → 進捗確認 → 優先事項 → 引継ぎ
   - すべて画面に表示されます
