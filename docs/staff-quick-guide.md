# 📋 申し送りシステム 3分ガイド

## 🎯 このシステムで変わること

| 今まで | これから |
|:---:|:---:|
| 📝 口頭申し送り | 💻 タイムライン記録 |
| 🤔 重要事項の記憶頼み | 🚨 自動アラート表示 |
| ✍️ 特記事項を一から手入力 | 🔄 重要申し送りが自動転記 |

---

## 🚀 覚えるのはこの3つだけ

### 1️⃣ 申し送りタイムライン
**いつ**: 気づいた時にその場で
※ 画面上部のメニュー「申し送りタイムライン」からも、いつでも直接開けます。
**何を**: 利用者の様子で気になったこと
**コツ**: 短文で、カテゴリと重要度を選ぶだけ

```
例: "田中さん、血圧高め。要観察"
カテゴリ: 体調  重要度: 重要
```

### 2️⃣ 朝会・夕会
**司会者**: MeetingGuide画面を開いて進行
**参加者**: 重要アラートを確認するだけ
**ポイント**: 赤いバッジがあったら要確認

### 3️⃣ 日次記録
**変化**: 特記事項に重要な申し送りが自動で下書き
**作業**: 不要な行を削除して、必要なら追記
**安心**: 自動で入った文章は自由に編集・削除OK

---

## 📱 画面の場所

| 機能 | URL | いつ使う |
|---|---|---|
| 申し送り | `/handoff-timeline` | 随時・気づいた時 |
| 朝会・夕会 | `/meeting-guide` | 8:50・16:30 |
| 日次記録 | `/daily-records` | 17:00頃 |

---

## ⚡ クイックスタート

### 初日にやること
1. 申し送りタイムラインを開く
2. 「新規作成」ボタンを押す
3. 何でもいいので1件入力してみる
4. 朝会または夕会で画面を見てみる

### 2日目から
- 気づいたことがあったらタイムラインに記録
- 重要なことは重要度「重要」に設定
- 日次記録で特記事項の自動転記を確認

---

## 🆘 困った時は

**「何を書けばいいかわからない」**
→ 利用者の名前 + 気づいたこと（短文）でOK

**「重要度の判定がわからない」**
→ 次のスタッフに絶対伝えたい = 重要、気にかけて = 要注意

**「システムが使えない」**
→ いつも通り口頭・紙で申し送り（併用でOK）

**「操作方法がわからない」**
→ [サポート担当者名・連絡先]まで
